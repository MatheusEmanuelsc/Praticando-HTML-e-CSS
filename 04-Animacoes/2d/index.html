🎨 Transformações 2D em CSS

Guia completo com exemplos práticos e comentários detalhados

🚀 Parte 1: Transformações Básicas

📍 1. Translate (Movimento)

A propriedade translate() move um elemento de sua posição original sem afetar outros elementos.

/* Sintaxe básica do translate */ transform: translate(x, y); /* Exemplos práticos */ .elemento { transform: translate(50px, 100px); /* Move 50px direita, 100px baixo */ transform: translateX(30px); /* Apenas eixo X */ transform: translateY(-20px); /* Apenas eixo Y (negativo = para cima) */ } /* Com transição suave */ .elemento { transition: transform 0.3s ease; }
HOVER

👆 Passe o mouse para ver o movimento!

📏 2. Scale (Escala)

A função scale() redimensiona elementos. Valores maiores que 1 aumentam, menores que 1 diminuem.

/* Sintaxe do scale */ transform: scale(valor); /* Escala uniforme */ transform: scale(x, y); /* Escala diferente para cada eixo */ /* Exemplos */ .aumentar { transform: scale(1.5); /* 150% do tamanho original */ } .diminuir { transform: scale(0.8); /* 80% do tamanho original */ } .esticar { transform: scale(2, 0.5); /* Largo x2, altura x0.5 */ }

👆 Círculo aumenta ao passar o mouse!

🔄 3. Rotate (Rotação)

A função rotate() gira elementos em torno de seu centro (ou ponto de origem definido).

/* Sintaxe da rotação */ transform: rotate(ângulo); /* Unidades aceitas */ transform: rotate(45deg); /* Graus (mais comum) */ transform: rotate(0.5turn); /* Voltas (0.5 = meia volta) */ transform: rotate(1.57rad); /* Radianos */ /* Exemplos práticos */ .girar-direita { transform: rotate(90deg); /* 90 graus horário */ } .girar-esquerda { transform: rotate(-45deg); /* 45 graus anti-horário */ }
GIRA

👆 Elemento gira 360° no hover!

💡 Dica Importante:

Todas essas transformações podem ser combinadas usando espaços: transform: translate(50px, 20px) rotate(45deg) scale(1.2);

🎯 Centrar Elementos

.centro { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }

📱 Efeito Botão

.botao:active { transform: scale(0.95); transition: transform 0.1s; }

⚡ Parte 2: Transformações Avançadas

🔀 4. Skew (Inclinação)

A função skew() inclina elementos, criando efeitos de perspectiva e distorção.

/* Sintaxe do skew */ transform: skew(x-angle, y-angle); transform: skewX(angle); /* Apenas eixo X */ transform: skewY(angle); /* Apenas eixo Y */ /* Exemplos */ .inclinar { transform: skew(20deg, 10deg); /* Inclina nos dois eixos */ } .paralela { transform: skewX(30deg); /* Efeito paralelo */ }
SKEW

👆 Elemento se inclina no hover!

🎯 5. Transform-Origin (Ponto de Origem)

Define o ponto ao redor do qual as transformações ocorrem. Por padrão é o centro (50% 50%).

/* Sintaxe do transform-origin */ transform-origin: x y; /* Valores possíveis */ transform-origin: center center; /* Padrão - centro */ transform-origin: top left; /* Canto superior esquerdo */ transform-origin: bottom right; /* Canto inferior direito */ transform-origin: 50px 100px; /* Posição específica */ transform-origin: 25% 75%; /* Porcentagem */ /* Exemplo prático */ .girar-canto { transform-origin: top left; transform: rotate(45deg); /* Gira pelo canto */ }
ORIGIN

👆 Rotaciona pelo canto superior esquerdo!

🧮 6. Matrix (Transformação Matricial)

A função matrix() combina todas as transformações 2D em uma única função matemática.

/* Sintaxe da matrix */ transform: matrix(a, b, c, d, e, f); /* Parâmetros: a, d = escala b, c = inclinação e, f = translação */ /* Equivalências */ transform: translate(50px, 100px); /* É igual a: */ transform: matrix(1, 0, 0, 1, 50, 100); /* Exemplo complexo */ .matrix-complexa { transform: matrix(1.2, 0.3, -0.3, 1.2, 30, 20); /* Combina escala, rotação e movimento */ }

👆 Transformação matricial complexa!

⚠️ Atenção:

A função matrix() é poderosa mas complexa. Para a maioria dos casos, use as funções individuais que são mais legíveis e fáceis de manter.

🔄 Flip Horizontal

.flip-h { transform: scaleX(-1); }

🔃 Flip Vertical

.flip-v { transform: scaleY(-1); }

🎭 Parte 3: Animações e Efeitos Avançados

🎬 7. Animações com @keyframes

Combine transformações com animações CSS para criar efeitos dinâmicos e atraentes.

/* Definindo keyframes para rotação contínua */ @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } /* Animação de pulo */ @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } } /* Pulsação */ @keyframes pulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.2); } } /* Aplicando as animações */ .elemento-girando { animation: spin 2s linear infinite; } .elemento-pulando { animation: bounce 1.5s ease-in-out infinite; }

👆 Animações contínuas: Girar, Pular, Pulsar

🎪 8. Efeitos 3D Simulados

Mesmo em 2D, podemos criar efeitos que simulam profundidade e dimensão.

/* Efeito de profundidade com perspective */ .container-3d { perspective: 1000px; /* Define a distância de visualização */ } .carta-3d { transform-style: preserve-3d; /* Preserva o espaço 3D */ transition: transform 0.3s ease; } .carta-3d:hover { /* Simula rotação 3D */ transform: perspective(1000px) rotateX(15deg) rotateY(15deg) translateZ(20px); } /* Sombra para reforçar o efeito */ .carta-3d { box-shadow: 0 10px 20px rgba(0,0,0,0.2); } .carta-3d:hover { box-shadow: 0 20px 40px rgba(0,0,0,0.3); /* Sombra maior no hover */ }

🎨 9. Combinações Avançadas

Combinando múltiplas transformações para criar efeitos únicos e impressionantes.

/* Combinação complexa de transformações */ .efeito-complexo { transition: transform 0.6s cubic-bezier(0.68, -0.55, 0.265, 1.55); } .efeito-complexo:hover { transform: translate(30px, -20px) /* Move para direita e cima */ rotate(180deg) /* Rotaciona meia volta */ scale(1.3) /* Aumenta o tamanho */ skew(10deg, 5deg); /* Adiciona inclinação */ } /* Efeito de "morphing" */ .morphing { border-radius: 50%; transition: all 0.5s ease; } .morphing:hover { border-radius: 20%; transform: rotate(45deg) scale(1.2); }

👆 Combinação de múltiplas transformações!

⚙️ 10. Propriedades de Controle

Propriedades que controlam como as transformações se comportam.

/* Timing functions - Controla a velocidade da transição */ .elemento { transition-timing-function: ease; /* Padrão */ transition-timing-function: linear; /* Velocidade constante */ transition-timing-function: ease-in; /* Acelera no início */ transition-timing-function: ease-out; /* Desacelera no final */ transition-timing-function: ease-in-out; /* Acelera e desacelera */ /* Curva personalizada (cubic-bezier) */ transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); } /* Delays - Atraso antes da transformação */ .com-delay { transition-delay: 0.2s; /* Espera 0.2s antes de iniciar */ } /* Duração da transição */ .rapido { transition-duration: 0.1s; } .lento { transition-duration: 2s; }

🎯 Dicas de Performance:

  • Use transform em vez de alterar top, left, width, height
  • Prefira transform e opacity para animações suaves
  • Use will-change: transform para elementos que serão animados
  • Evite animar muitos elementos simultaneamente

📱 Responsividade:
