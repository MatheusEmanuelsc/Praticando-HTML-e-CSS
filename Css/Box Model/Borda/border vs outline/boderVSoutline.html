Neste exemplo, adicionamos um bloco de conteúdo (div.content) dentro da caixa para destacar o comportamento das propriedades border e outline. As caixas coloridas (div.border-example e div.outline-example) estão dentro da mesma caixa para comparar os efeitos.

visualizar esta página no navegador, você poderá observar que o border afeta o layout, aumentando o tamanho da caixa, enquanto o outline não afeta o layout e é desenhado sobre o conteúdo. O outline é muitas vezes usado para destacar elementos sem afetar o tamanho total da caixa.
